## To run the app

```java

mvn clean install
// run the either of the follwing command
mvn clean compile exec:java

mvn exec:java -Dexec.mainClass="com.imperionite.inventorysystem.InventorySystem"
//  prefer to use the properties
mvn exec:java

```
